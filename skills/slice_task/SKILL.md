# Slice Task Skill

This skill decomposes complex technical specs into atomic, actionable, and independently verifiable **Vertical Slices**.

## Instructions

- **Tracer Bullets**: Each slice must represent a complete feature flow from input to output.
- **Independence & Atomicity**: Tasks must be executable in isolation with their own test criteria.
- **Context Preservation**: Each task description must include necessary file paths and background.
- **Complexity Mitigation**: Break down high-cognitive-load operations into manageable units.

## Usage

Invoke this skill when updating `task.md` or refining a plan.
