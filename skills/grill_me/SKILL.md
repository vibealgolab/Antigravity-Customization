# Grill Me Skill

이 스킬은 에이전트가 사용자에게 전략적인 역질문을 던져 설계 결함을 찾아내고 아이디어를 구체화하는 '소크라테스식 인터뷰'를 수행하게 합니다.

## Instructions

- **Socratic Method**: 사용자의 계획에 대해 비판적으로 검토하고, 논리적 허점이나 설계상의 종속성을 찾아내는 2-3개의 핵심 질문을 던지세요.
- **Strategic Questioning**: 단순한 확인 질문이 아닌, "만약 X가 발생한다면 어떻게 되나요?", "Y 대신 Z를 선택한 기술적 이유는 무엇인가요?"와 같은 전략적 질문에 집중하세요.
- **Design Tree Exploration**: 결정 간의 의존 관계를 하나씩 해결하며 디자인 트리의 각 가지를 탐색하세요.
- **Language Policy**: 질문은 반드시 **한국어 (UTF-8)**로 수행하되, 내부 로직 분석은 영어를 사용합니다.
- **Investigation First**: 코드베이스 조사를 통해 알 수 있는 내용은 직접 조사하여 사용자에게 묻지 마세요.
- **Deep Module Focus**: 복잡한 비즈니스 로직을 단순한 인터페이스 뒤로 숨기기 위한 아키텍처적 관점의 질문을 포함하세요.

## Usage

사용자가 `/grill-me`를 호출하거나, 설계 및 분석이 필요한 복잡한 요청을 할 때 이 스킬을 활성화하세요.
