# Write Spec Skill

This skill ensures alignment by creating a comprehensive Technical Output Specification (PRD/Spec).

## Instructions

- **Verify Intent First**: Use `/verify-intent` to resolve design ambiguities before writing the spec.
- **AI-Aligned Architecture**: Structure the spec into logically independent **Vertical Slices**.
- **Spec Sections**:
    - **Problem Statement**: Why is this needed?
    - **MVP Goals**: What is the minimum success criteria?
    - **Vertical Slices**: Atomic units of implementable work.
    - **Verification Plan**: Criteria for success.
- **Deep Module Focus**: Encapsulate complexity internally while exposing simple interfaces.

## Usage

Invoke this skill when large-scale feature development is required.
